Summary (Penn State)

Proteins can fold during their synthesis by ribosome through the process of co-translational folding. Such co-translational folding occurs out of equilibrium, meaning thereby the speed of translation can drastically influence the ability of proteins to fold and function. Changes to translation speed can prevent folding, increase misfolding, can cause aggregate formation, and are implicated as causal factors in cystic fibrosis, hemophilia, and certain cancers. Most protein structure prediction methods do not consider the biological process of translation when making predictions and thus neglect the influence of translation speed. I am working on how different combinations of molecules (Amino acid, tRNA, and Codon) present at A-site, P-site, and E-site of ribosome influence the translation elongation rate at A-site through the ribosome profiling dataset. We have developed a new method to see how P and E site molecules show the synergetic effect at the A-site to regulate the speed. I have developed a pipeline in Python for analysis (GitHub). This pipeline contains multiple steps.

  1. Raw fastq file - Download fastq file from SRA server
  2. Processed fastq file - Filter out low-quality reads. Many tools are available to filter the process. We used Standard processed which is given into respected research paper
  3. Alignment - The processed reads were first aligned to the ribosomal RNA sequences using Bowtie 2 (v2.2.5). The unaligned reads were then aligned to the Saccharomyces cerevisiae assembly (UCSC: sacCer3) using Tophat (v2.1.1)
  4. A-site profile - A-site profile generated through in-house generated webtool 'Ribo-Seq A-site Identifier' (https://a-site.vmhost.psu.edu/)
  5. Robust analysis in multiple datasets - This part is done by an in-house generated script.

Simultaneously, I have generated data for a machine learning project for biophysical model development of amino acid and tRNA effects on translation-elongation speed.

Henceforth, creating theoretical spectra from the database and reading spectra file that will be common part and after peptide identification, we are integrating the combined FDR process.
